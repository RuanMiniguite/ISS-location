## About the project

<div align="center">
  <img src="/assets/Desktop.png" width=80%>
<br><br>

  [![Site][Site-shields]][Site-link]
</div>

<p>Site shows ISS (International Space Station) telemetry in real time, in addition to displaying its geographical position on the map.</p>


<h4> Telemetry: </h4>

- Latitude
- Longitude
- Altitude
- Speed
- Hour
- Visibility
